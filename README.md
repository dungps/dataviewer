# Dataviewer

This is a database management like TablePlus or DataGrip but it free

# Development

## Requirement

- [Bun 1.1+](https://bun.sh)
- [Go 1.21+](https://go.dev)
- MacOS 13+

## Start Development
```shell
make run
```

## Build assets
```
bun build index.ts
```

## Build application
```
make build
```
